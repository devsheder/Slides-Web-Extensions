# demo

## Requirements

- Install pnpm (or prefix every pnpm command with npx, example: `pnpm i` become `npx pnpm i`)

## start demo app

```bash
cd demo/lego-website
pnpm i
pnpm start
```

## build de web extension

```bash
cd demo/demo-logs
pnpm i
pnpm build
pnpm start:firefox # or pnpm start:chrome
```
